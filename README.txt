# Leia-me - Projeto em React Native usando Expo

Este projeto é desenvolvido usando React Native e Expo. Ele fornece uma base para criar aplicativos móveis multiplataforma.

## Pré-requisitos

Certifique-se de ter as seguintes ferramentas instaladas em sua máquina:

- Node.js: [https://nodejs.org](https://nodejs.org)
- Expo CLI: Você pode instalá-lo executando o seguinte comando no terminal:

npm install -g expo-cli
